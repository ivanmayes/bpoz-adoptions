Big Paws Adoptable Dogs - Iframe Example

This page demonstrates how to embed the adoptable dogs page in an iframe on your website.

How to Embed

Copy this code to embed the adoptable dogs page on your website:

<iframe src="https://your-domain.vercel.app/adoptable-dogs/" title="Big Paws Adoptable Dogs" width="100%" height="800" style="border: none;" loading="lazy" allowfullscreen> </iframe>

Responsive Iframe

For a responsive iframe that maintains aspect ratio:

<div style="position: relative; padding-bottom: 75%; height: 0; overflow: hidden;"> <iframe src="https://your-domain.vercel.app/adoptable-dogs/" title="Big Paws Adoptable Dogs" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none;" loading="lazy" allowfullscreen> </iframe> </div>

Advanced Options

You can also link directly to specific features:

  • With AI Chat Open: /adoptable-dogs/?openAIBar=true
  • Specific Dog: /adoptable-dogs/dog-detail.html?id=123
